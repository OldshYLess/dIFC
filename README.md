# dIFC
Respository for deep imaging flow cytometry (dIFC).
## Installation
### Clone this repo
`git clone https://github.com/OldshYLess/dIFC`
### Install PyTorch and other dependencies
* PyTorch--------  v1.5.0
* OpenCV---------  v3.4.1
* Numpy----------  v1.19.2
* PIL------------  v8.2.0
